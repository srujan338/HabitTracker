# Quickstart: Habit Tracker

## Prerequisites
- Python 3.10+
- Streamlit

## Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install streamlit
   ```

## Running the Application
Run the Streamlit app from the root directory:
```bash
streamlit run main.py
```

## User Guide
1. **Add a Habit**: Type a name in the "New Habit Name" field and click "Add".
2. **Track Progress**: Find your habit in the list and click "Mark Complete".
3. **View Stats**: Your current and longest streaks update automatically.
4. **Clean Up**: Use the delete button to remove habits you no longer need.
